# 某些有助于精简语法的语法特性
## 切片

1. 用途：用于快速地获取 list 或者 tuple 部分元素的操作
2. 语法：

    ```
    NAME[num1: num2: num3]
    ```

    其中，NAME 是 list 或者 tuple 的名称，括号使用**方括号**；
